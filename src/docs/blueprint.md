# **App Name**: GoldChic Dating

## Core Features:

- User Profile Creation: Allow users to create detailed profiles with photos and interests.
- Profile Swiping: Enable users to swipe through profiles (like/dislike).
- User Matching: Match users based on mutual likes.
- Real-time Chat: Allow matched users to chat in real-time.
- Location-based Suggestions: Suggest potential matches based on the user's current location.
- AI Match Enhancement Tool: AI analyzes user profiles to suggest potential matches, and can choose whether or not to include people in a potential match based on shared hobbies and interests.

## Style Guidelines:

- Primary color: Gold (#FFD700) for luxury and exclusivity.
- Background color: Black (#121212) for a sleek and modern look.
- Accent color: Off-white (#F0F0F0) for text and interactive elements.
- Body and headline font: 'Poppins', a geometric sans-serif font for a precise, contemporary, fashionable feel.
- Use simple and elegant icons in gold and white.
- Clean and modern layout with a focus on user profiles.
- Subtle animations for swipes and transitions.
